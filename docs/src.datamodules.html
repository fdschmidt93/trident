class src.datamodules.base.BaseDataModule(*args: Any, **kwargs: Any)

Bases: pytorch_lightning., abc.ABC

The base class for all datamodules.

Parameters

  • collate_fn (omegaconf.dictconfig.DictConfig) –

    Needs to return a Callable that processes a batch returned by the DataLoader.

    See also:

    src.modules.base.TridentModule.forward()
    src.modules.base.TridentModule.training_step()
    Example: model of sequence classification:

  • batch_size (int) – The batch size returned by your DataLoader | See also: Pytorch DataLoader <https://pytorch.org/docs/stable/data.html#torch.utils.data.DataLoader>

  • num_workers (int) – The number of workers for your DataLoader | See also: Pytorch DataLoader <https://pytorch.org/docs/stable/data.html#torch.utils.data.DataLoader>

  • train_collate_fn (omegaconf.dictconfig.DictConfig, optional) – If passed, replaces collate_fn for the train dataloader.

  • val_collate_fn (omegaconf.dictconfig.DictConfig, optional) – If passed, replaces collate_fn for the val dataloader.

  • test_collate_fn (omegaconf.dictconfig.DictConfig, optional) – If passed, replaces collate_fn for the test dataloader.

  • seed (int, optional) – Linked against config.seed by default for convenience and maybe used for functionality that is not yet set by pytorch_lightning.seed_everything, which sets the seed for pytorch, numpy and python.random.

Sets up data_{train, val, test} datasets that are fed to the corresponding DataLoader instances.

Typically wraps datasets in setup method of dataset.

Example: MNLI
def setup(self, stage: Optional[str] = None):
        if stage in (None, "fit"):
            dataset = load_dataset("glue", "mnli")
            dataset = dataset.map(self.preprocess, num_proc=cpu_count())
            dataset = dataset.rename_column("label", "labels")
            self.data_train = dataset["train"]
            self.data_val = concatenate_datasets(
                [dataset["validation_mismatched"], dataset["validation_matched"]]
            )
            # if stage in (None, "test"):
            self.data_test = self.data_val
