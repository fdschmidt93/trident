class trident.core.dataset.TridentDataset(cfg: omegaconf.dictconfig.DictConfig)[source]¶

Bases: object

A container class for handling one or many datasets in Trident workflows.

This class offers a unified approach for managing datasets created from trident-style Hydra configurations. It provides consistent and ergonomic access whether the encapsulated data comprises a single dataset or multiple datasets.

Key Features: - Encapsulates one or many datasets derived from a Hydra configuration. - Allows access to datasets using either integer indices or string keys. - Internally represents a single dataset with a None key. - Abstracts the underlying data structure for seamless access.

Parameters
